# Brand Spec: FATEC SP (Reconciled with PostHog Playful)

## Visual Direction
A playful, student-focused healthcheck dashboard. It combines the technical authority of FATEC with the irreverent, approachable personality of the PostHog design system.

## Color Tokens (OKLch)
- `--bg`:      oklch(98% 0.004 95)    /* PostHog Warm Parchment */
- `--surface`: oklch(94% 0.01 95)     /* Sage Cream */
- `--fg`:      oklch(20% 0.018 70)    /* Olive Ink (Primary Text) */
- `--muted`:   oklch(48% 0.012 70)    /* Muted Olive */
- `--border`:  oklch(85% 0.01 95)     /* Sage Border */
- `--accent`:  oklch(55% 0.22 25)     /* FATEC Deep Red (Accent) */

## Typography
- **Display**: IBM Plex Sans (Weight 800) - Assertive and technical.
- **Body**: IBM Plex Sans (Weight 400) - Highly readable.
- **Mono**: Source Code Pro - For status codes and server logs.

## Layout & Posture
- **Playful Mascot**: A hedgehog "technician" checking the servers.
- **Status Indicator**: Large, simple "Status: UP" or "Status: DOWN" indicators.
- **Tone**: Simple, direct, and slightly playful (e.g., "The college has power! ⚡" vs "Server is unreachable").
- **Constraints**: Responsive web (Mobile + Desktop).
